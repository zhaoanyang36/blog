COVID-19 Occurrence

Daily Report Map  

This application shows the number of COVID-19 instances recorded by Johns Hopkins University in each county/province, automatically updated daily. The color intensities and sizes of the circles represent the number of cases.

To get more information about any specific county/province/country, click on the respective circle on the map.

To see detailed information for the United States, click on the button below.

The 2018 ACS 5-Year Census population data can be displayed using the buttons below. It takes a few seconds to download the data at the back from the Census API so be patient. The total population can demonstrate whether the outbreak correlates with the population. The percentage of the elderly population in each county can show which counties might be more vulnerable to the pandemic. These results can be used as a starting point for further research.
